# Spreelo V135 — Calendar readability and chronology

## Changes
- Increased all calendar content text to the same comfortable scale used in AI Innehållsstudio.
- Campaign title, description, metadata, filter controls, detail text, tags and buttons are larger.
- Default campaign order is chronological: closest upcoming campaign first.
- The first chronological campaign is expanded when the page opens.
- Sort control now offers closest first or latest first.
- Planned status is green.
- Current status is orange/peach so it cannot be confused with planned.

## Unchanged
- No AI calls were added.
- No campaign planning logic changed.
- No credit logic changed.
- No database or SQL changes are required.
- Campaign handoff to AI Innehållsstudio is unchanged.
