# Spreelo V137 — Responsive calendar controls

Presentation-only calendar update based on V136.

## Changed
- Prevents the sorting dropdown from overlapping the campaign filters.
- Uses a stable two-column controls layout on landscape tablets and narrow desktops.
- Stacks filters and sorting in the right column beside the date card.
- Uses a compact horizontal date card on smaller tablets and phones.
- Makes the expanded filter/date area full-width with consistent spacing and less empty grey space.
- Keeps filter buttons and the sorting dropdown within their own cards at all supported widths.

## Unchanged
- No campaign planning logic changes.
- No AI calls added or changed.
- No credit, database, campaign-policy, product-format or handoff changes.
- No SQL migration required.
